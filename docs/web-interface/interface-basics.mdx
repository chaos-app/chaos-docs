# Interface basics

## Account menu

Click on the account menu to customize your profile, go back to your main collections page, or log out of your account.

![](/img/interface-basics/interface-account.png)

## Help panel

Click on the question mark icon on the bottom-right corner of the app to bring up the Help panel, where you can find out about the latest updates to Chaos, or send us a message.

![](/img/interface-basics/interface-help.png)

## Sidebar toggle

You can show/hide the sidebar by clicking the sidebar toggle.

![](/img/interface-basics/interface-sidebar.png)

## Color mode toggle

You can switch between light and dark color modes by clicking the color mode toggle.

![](/img/interface-basics/interface-dark-mode.png)
